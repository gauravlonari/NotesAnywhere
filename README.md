# NotesAnywhere
 Online Notes Keeping Website made using Javascript!

### Stack used: MERN

## Functional Components:
1. Login and Logout
2. Sign in and Register
2. Homepage
2. Notes Dashboard
2. Add Notes
2. Delete Notes
2. Update Notes
2. Profile
2. Change Password
2. About
